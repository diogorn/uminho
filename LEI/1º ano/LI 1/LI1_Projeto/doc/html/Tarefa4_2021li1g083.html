correrMovimentos :: Jogo -> [Movimento] -> Jogo #

A função "correrMovimentos" prepara uma série de movimentos para serem aplicados ao Jogador.

moveJogador :: Jogo -> Movimento -> Jogo #

"moveJogador" é auxiliar de "correrMovimentos" e aplica um movimento especifico ao jogador. Esta função também identifica se o personagem carrega ou não uma Caixa.

checkPeca :: Mapa -> Coordenadas -> Coordenadas -> Peca #

A função "checkPeca" identifica o tipo de peça presente numa coordenada especifica.

moveLeftFalse :: Mapa -> Jogador -> Jogo #

Caso o jogador não carregue uma Caixa:

"moveLeftFalse" é a função responsável por movimentar o jogador para a esquerda quando este não carrega uma caixa.

moveRightFalse :: Mapa -> Jogador -> Jogo #

"moveRightFalse" movimenta o jogador para a direita quando este não carrega uma caixa.

moveUpFalse :: Mapa -> Jogador -> Jogo #

"moveUpFalse" permite ao jogador trepar blocos ou caixas.

interCaixaFalse :: Mapa -> Jogador -> Jogo #

"interCaixaFalse" é responsável pela interação Jogador - Caixa, ou seja, caso este interaja com uma caixa, passará a carregá-la se possivel.

moveLeftTrue :: Mapa -> Jogador -> Jogo #

Caso o jogador carregue uma Caixa:

"moveLeftTrue" é a função responsável por movimentar o jogador e a caixa que este carrega para a esquerda.

moveRightTrue :: Mapa -> Jogador -> Jogo #

"moveRightTrue" movimenta o jogador e a caixa que este carrega para a direita.

moveUpTrue :: Mapa -> Jogador -> Jogo #

"moveUpTrue" permite ao jogador trepar blocos ou caixas se estiver a carregar uma caixa.

interCaixaTrue :: Mapa -> Jogador -> Jogo #

"interCaixaTrue" é responsável pela interação Jogador - Caixa, ou seja, como o jogador já carrega uma caixa, se este tentar apanhar outra, não irá conseguir.

addremoveCaixa :: Mapa -> Coordenadas -> Coordenadas -> Mapa #

A "addremoveCaixa" elimina a caixa de uma coordenada sempre que um movimento/interação é executado, de modo a não existir a mesma caixa em posições diferentes
